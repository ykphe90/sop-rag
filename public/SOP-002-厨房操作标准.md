# SOP-002: 厨房操作标准作业程序

**文件编号**: SOP-002  
**版本**: v1.8  
**生效日期**: 2024-03-01  
**适用范围**: 全门店厨房  
**负责部门**: 运营部  

---

## 1. 目的

规范厨房日常操作流程，确保出品质量一致、效率达标，并保障员工操作安全。

## 2. 开店准备流程（营业前 90 分钟）

### 2.1 设备检查（负责人：当班厨师长）
1. 检查所有冷藏/冷冻设备温度并记录
2. 开启抽油烟机，确认运作正常
3. 检查燃气管路，确认无泄漏（使用肥皂水测试）
4. 开启蒸笼预热（设定 100°C，预热时间约 20 分钟）
5. 检查消防设备（灭火器压力表在绿色区域）

### 2.2 食材准备（Mise en Place）
- 根据当日预估营业量准备食材
- 参考《每日食材准备量标准表》（SOP-002-附件A）
- 预估依据：
  - 工作日午餐：约 150-200 份
  - 工作日晚餐：约 200-280 份
  - 周末/假日：上调 30-50%
  - 特殊节日（农历新年、开斋节等）：按历史数据另行通知

### 2.3 调味料与酱汁
- 每日开店前确认以下酱汁库存充足：
  - 黑醋姜丝酱（每日新鲜调配，保质期 8 小时）
  - 辣油（每周制作两次，周一和周四）
  - 蒜蓉酱油（每日新鲜调配）
- 酱汁调配须严格按照《标准配方卡》执行

## 3. 出品标准

### 3.1 小笼包制作标准

| 项目 | 标准 |
|-----|------|
| 皮的厚度 | 1.5-2mm |
| 馅料重量 | 16g ± 1g |
| 褶数 | 18 褶 |
| 蒸制时间 | 4 分 30 秒（从水开始沸腾计时） |
| 蒸制温度 | 100°C |
| 每笼数量 | 10 颗 |
| 出品外观 | 皮薄透亮，褶花均匀，无破损漏汤 |

### 3.2 炒饭制作标准

| 项目 | 标准 |
|-----|------|
| 米饭 | 隔夜饭，粒粒分明 |
| 用油量 | 30ml |
| 炒制时间 | 2-3 分钟 |
| 锅温 | 200-230°C |
| 蛋液 | 40g，先炒至半熟 |
| 调味 | 盐 3g，白胡椒粉 1g，酱油 5ml |
| 出品标准 | 颜色金黄均匀，无结块，米粒完整 |

### 3.3 酸辣汤制作标准

| 项目 | 标准 |
|-----|------|
| 汤底 | 使用当日熬制的高汤 |
| 醋量 | 每份 15ml 黑醋 |
| 辣油 | 每份 5ml |
| 勾芡 | 适度挂勺，不过稠 |
| 蛋花 | 均匀细丝状 |
| 出品温度 | ≥ 85°C |
| 每份容量 | 300ml |

## 4. 出餐流程

### 4.1 出餐顺序原则
1. 先冷后热（冷菜 → 汤品 → 蒸品 → 炒菜 → 主食 → 甜品）
2. 同桌菜品间隔不超过 5 分钟
3. 蒸品出笼后须在 30 秒内送达餐桌

### 4.2 出餐时间标准

| 菜品类别 | 从下单到出餐 |
|---------|------------|
| 冷菜/小菜 | ≤ 5 分钟 |
| 蒸品（小笼包等） | ≤ 8 分钟 |
| 炒菜 | ≤ 10 分钟 |
| 汤品 | ≤ 6 分钟 |
| 甜品 | ≤ 5 分钟 |

### 4.3 高峰期管理
- 午餐高峰（11:30-13:30）和晚餐高峰（18:00-20:00）
- 高峰期额外措施：
  - 提前备好热门菜品的半成品
  - 蒸笼保持满负荷运转
  - 安排专人负责传菜
  - 每 15 分钟检查一次排队情况

## 5. 特殊饮食处理

### 5.1 过敏原管理
常见过敏原标注（须在菜单上明确标示）：
- 麸质（小麦）：小笼包皮、面条、馒头
- 甲壳类：虾饺、海鲜炒饭
- 花生：部分凉菜
- 芝麻：芝麻包、部分酱汁
- 鸡蛋：蛋炒饭、酸辣汤、蛋糕类
- 大豆：酱油（几乎所有菜品）

### 5.2 过敏原处理程序
1. 服务员接到过敏通知后，在订单上标注红色"ALLERGY"标记
2. 厨师长亲自确认可替代方案
3. 使用独立、已消毒的锅具和器具准备
4. 出品时使用红色标识碟
5. 由厨师长确认后方可出餐

### 5.3 素食/清真处理
- 素食菜品须使用专用锅具（标记绿色手柄）
- 清真要求须使用清真认证食材并分区处理
- 不确定时，主动询问顾客具体需求

## 6. 食材浪费管理

### 6.1 浪费监控
- 每日记录食材浪费量（按类别）
- 每周汇总分析，浪费率目标：≤ 3%
- 浪费率计算：(废弃食材成本 / 总食材成本) × 100%

### 6.2 减少浪费措施
- 精确预估每日用量（参考 POS 历史数据）
- 边角料再利用（如蔬菜边角料用于高汤）
- 接近保质期食材优先使用
- 每月进行一次浪费原因分析会议

## 7. 闭店流程

### 7.1 清洁（最后一桌客人离开后）
1. 关闭所有炉灶和燃气总阀
2. 清洗所有锅具、器具
3. 清洁所有工作台面和设备表面
4. 清扫和拖洗地板
5. 清理并更换垃圾袋
6. 清洗排水沟

### 7.2 食材处理
1. 剩余半成品正确标注日期并冷藏
2. 检查所有食材的保质期
3. 记录需要补充的食材到《采购申请表》
4. 确认冷藏/冷冻设备温度正常并记录

### 7.3 安全检查
1. 确认燃气总阀关闭
2. 确认所有电器设备关闭（冷藏/冷冻除外）
3. 检查门窗锁好
4. 开启夜间安保系统
5. 当班厨师长签名确认《闭店检查表》
